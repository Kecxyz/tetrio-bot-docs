# `"type": "keyup"`

Denoting which key was lifted.

## Formats

* (string) `type`: `"keyup"`
  * (obj) `data`:
    * (string) `key`: Possible values are `"hardDrop"`, `"softDrop"`, `"moveLeft"`, `"moveRight"`, `"rotateCW"`, `"rotateCCW"`, `"rotate180"` and `"hold"`.
    * (float) `subframe`: The subframe the action was done on.
